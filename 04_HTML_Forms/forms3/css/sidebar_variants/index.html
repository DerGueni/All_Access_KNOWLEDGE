Verwendung

Um eine Variante zu verwenden, fuegen Sie den entsprechenden Link in shell.html ein:

<!-- In shell.html nach dem <style>-Block einfuegen: -->

<link rel="stylesheet" href="css/sidebar_variants/sidebar_variant_09_glass.css">

<!-- ODER fuer Colorful Icons: -->
<link rel="stylesheet" href="css/sidebar_variants/sidebar_variant_10_colorful.css">

Hinweis: Die Varianten-CSS ueberschreibt die Standard-Sidebar-Styles. Reihenfolge der <link>-Tags beachten!
