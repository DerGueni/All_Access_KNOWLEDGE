Unified Header - Button-Referenz

1. Standard-Buttons (Funktionsbezogen)

Jeder Button hat eine feste Farbe basierend auf seiner Funktion. Diese Farben sind in allen Formularen identisch.

Aktualisieren
Blau-Grün
<button class="form-btn form-btn-refresh">Aktualisieren</button>
Verwendung: Daten neu laden, Formular aktualisieren
Shortcut: Strg+R
Neu
Grün
<button class="form-btn form-btn-new">Neu</button>
Verwendung: Neuen Datensatz erstellen
Shortcut: Strg+N
Kopieren
Gelb
<button class="form-btn form-btn-copy">Kopieren</button>
Verwendung: Aktuellen Datensatz duplizieren
Shortcut: Strg+K
Speichern (Primär)
Blau (Primär)
<button class="form-btn form-btn-save">Speichern</button>
Verwendung: Änderungen speichern
Shortcut: Strg+S
Löschen (Gefährlich)
Rot
<button class="form-btn form-btn-delete">Löschen</button>
Verwendung: Datensatz löschen (destruktive Aktion!)
Shortcut: Strg+D
Schließen (Neutral)
Grau
<button class="form-btn form-btn-close">Schließen</button>
Verwendung: Formular schließen
Shortcut: Esc

6. Responsive Verhalten

Buttons passen sich automatisch an die Bildschirmgröße an.

Breakpoint Button-Höhe Font-Size Besonderheit
>1280px (Desktop) 32px 12px Volle Darstellung
1024px - 1280px (Tablet) 28px 11px Kompakter
<768px (Mobile) 26px 10px Icon-Only (Text versteckt)

7. Code-Beispiele

HTML (Direkt)

<div class="form-header-unified">
    <h1 class="form-title-unified">Auftragsverwaltung</h1>
    <div class="form-header-controls-unified">
        <button class="form-btn form-btn-refresh" onclick="refreshData()">Aktualisieren</button>
        <button class="form-btn form-btn-new" onclick="newRecord()">Neu</button>
        <button class="form-btn form-btn-delete" onclick="deleteRecord()">Löschen</button>
    </div>
</div>

JavaScript (Dynamisch)

// Option 1: Neue Instanz erstellen
const header = new UnifiedHeader({
    title: 'Auftragsverwaltung',
    buttons: ['refresh', 'new', 'delete'],
    onRefresh: () => loadData(),
    onNew: () => createRecord(),
    onDelete: () => deleteRecord()
});

// Option 2: Bestehenden Header verwenden
const header = attachUnifiedHeader('.form-header-unified');
header.on('refresh', () => loadData());
header.on('new', () => createRecord());

// Button-State ändern
header.setButtonEnabled('delete', false);
header.setButtonLabel('refresh', 'Lädt...');
header.setStatus('Aktiv', 'active');
