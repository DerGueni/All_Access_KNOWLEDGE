ANPASSUNGEN:
------------

Falls Sie Farben ändern möchten:

1. Öffnen Sie: create_design_variants.py
2. Suchen Sie die CSS-Konstanten:
   - CSS_MINIMALIST (Zeile ~60)
   - CSS_NORD (Zeile ~800)
3. Ändern Sie die Hex-Farbwerte
4. Führen Sie das Script erneut aus


WEITERE VARIANTEN:
------------------

Sie können beliebig viele Varianten erstellen:

1. Kopieren Sie eine CSS-Konstante (z.B. CSS_NORD)
2. Benennen Sie sie um (z.B. CSS_DRACULA)
3. Passen Sie die Farben an
4. Fügen Sie Code hinzu um die neue Variante zu erstellen
5. Script erneut ausführen
