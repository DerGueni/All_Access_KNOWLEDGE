# CONSYS Slash Commands - Verwendung

## Richtige Syntax für Claude Code CLI

**WICHTIG:** Project-Commands werden mit `/project:` Prefix aufgerufen!

| Befehl | Aufruf |
|--------|--------|
| button_audit | `/project:button_audit` |
| skills_an | `/project:skills_an` |
| skills_aus | `/project:skills_aus` |
| skills | `/project:skills` |
| skill [name] | `/project:skill consys-button-fixer` |

## Beispiele

```bash
# Skills aktivieren
/project:skills_an

# Button-Audit starten
/project:button_audit

# Spezifischen Skill laden
/project:skill consys-button-fixer
```

## Tipp
Tippe `/project:` und drücke TAB für Autocomplete.
